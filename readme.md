## Usage

Simply open `index.html` in your browser or deploy to GitHub Pages.

### Personalization

Add a name parameter to the URL to personalize the greeting:
```
https://yourusername.github.io/valentine?name=Adi
```

This will change "Hi dear," to "Hi Adi,"

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and animations
- `script.js` - Interactive functionality
- `not-cute-anymore.mp3` - Background music (you need to add this file)

## Setup for GitHub Pages

1. Upload all files to your GitHub repository
2. Make sure to include the MP3 file named `not-cute-anymore.mp3`
3. Enable GitHub Pages in repository settings
4. Your site will be live at `https://yourusername.github.io/repositoryname`
